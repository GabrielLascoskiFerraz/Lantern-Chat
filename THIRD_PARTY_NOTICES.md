# Avisos de terceiros

## Microsoft Fluent Emoji

O Lantern distribui imagens do Microsoft Fluent Emoji para a apresentação de emojis no Windows.
Os ativos originais são disponibilizados pela Microsoft sob a licença MIT no projeto
[`microsoft/fluentui-emoji`](https://github.com/microsoft/fluentui-emoji).

O pacote de distribuição usado no processo de compilação é
`@lobehub/fluent-emoji-3d`, também disponibilizado sob a licença MIT.
